# コーナー投票ボード（個人用・静的サイト）

**赤コーナー**と**青コーナー**の2択のみ。一口 **10万円** とし、**購入口数（手入力）**から投票シェアと人気順（1位・2位・同率）を表示します。データはブラウザ内に保存され、サーバーに送信しません。

画面は **1920×1080** を想定し、メイン枠の高さは最大 **1080px**・幅は最大 **1200px** 程度に収まるようレイアウトしています。背景画像は `assets/background.png`（差し替え可）です。

## 機能（v2）

- 現在時刻・ストップウォッチ・カウントダウン
- 赤／青の購入口数 → 購入金額（口 × 10万円）、合計口数・合計金額
- **人気バッジ**（1位・2位・同率1位）と **割合バー**、％表示
- JSON のエクスポート／インポート

公開 URL の例: `https://kurobe2240.github.io/-/`

## GitHub Pages

リポジトリの **Settings → Pages** で `main` の **`/(root)`** を指定。

ローカル確認: `index.html` をブラウザで開く。

## 注意

個人利用・学習用。表示は購入口数ベースのイメージであり、実際の投票・配当を保証するものではありません。
