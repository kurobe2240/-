# レース情報ボード（個人用・静的サイト）

ブラウザだけで動く手入力用のオッズ表・タイマーです。データはサーバーに送信されません。

## GitHub Pages で無料公開する手順

1. [GitHub](https://github.com) で新しいリポジトリを作成する（空でよい）。
2. このフォルダ内のファイルをリポジトリの**ルート**にプッシュする。
3. リポジトリの **Settings → Pages** を開く。
4. **Build and deployment** で **Source** を **Deploy from a branch** にし、**Branch** を `main`（または `master`）の **`/ (root)`** に設定して保存する。
5. 1〜2分後に `https://あなたのユーザー名.github.io/リポジトリ名/` で表示されます。

ローカルで確認する場合: `index.html` をブラウザで開くだけで動きます。

## 注意

- オッズ・人気は自分で入力する前提です。公表データの利用は各提供元の利用規約に従ってください。
- 配当・勝率の表示は簡易計算であり、実際の控除率やルールとは一致しません。
