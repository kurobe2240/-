# レース情報ボード（個人用・静的サイト）

ブラウザだけで動く手入力用のオッズ表・タイマーです。**データはサーバーに送信されません**（`localStorage` に自動保存）。

## 機能（v1.1）

- 現在時刻・ストップウォッチ・カウントダウン
- オッズ表（100円あたり払戻の目安・人気順ソート）
- 払戻計算・単純な勝率イメージ（正規化）
- **ブラウザへの自動保存**（再読み込み後も表が残る）
- **JSON をエクスポート / インポート**（バックアップ・別PCへ移すとき）

公開 URL の例: `https://kurobe2240.github.io/-/`

## GitHub Pages で無料公開する手順

1. このフォルダの内容をリポジトリの**ルート**にプッシュする。
2. **Settings → Pages** で **Branch** を `main` の **`/ (root)`** に設定して保存する。
3. 1〜2分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で表示されます。

ローカル確認: `index.html` をブラウザで開く。

## 注意

- オッズ・人気は自分で入力する前提です。公表データの利用は各提供元の利用規約に従ってください。
- 配当・勝率の表示は簡易計算であり、実際の控除率やルールとは一致しません。
